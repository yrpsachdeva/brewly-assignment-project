## 🚀 Features

| Feature | Details |
|---|---|
| 🏠 **Landing Page** | Hero section, featured menu, about & contact |
| ☕ **Full Menu Page** | 20+ drinks with filter tabs by category |
| 📱 **Fully Responsive** | Works beautifully on mobile, tablet & desktop |
| 🎨 **Premium Design** | Warm coffee-toned palette with bokeh effects |
| ⚡ **Smooth Navigation** | Sticky nav, smooth scroll, active link states |
| 🛒 **Interactive Cards** | Add to cart feedback, wishlist icons |
| 🍔 **Hamburger Menu** | Mobile-friendly collapsible nav |

---

## 📱 Mobile & Desktop Ready

This website is **fully responsive** — it looks and works great on **all screen sizes**:

- ✅ Mobile phones (320px+)
- ✅ Tablets (768px+)
- ✅ Laptops & Desktops (1024px+)

Built with **CSS media queries** and a **fluid grid layout** that adapts automatically.

```
brewly-website/
│
├── index.html       # 🏠 Landing page (hero, menu preview, about, contact)

```

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Flexbox, Grid, Animations
- **Vanilla JavaScript** — Menu filter, cart feedback, hamburger toggle
- **Google Fonts** — Playfair Display + Lato
- **Netlify** — Free hosting with auto-deploy

---

## 🏃 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/brewly-website.git

# 2. Open in browser
cd brewly-website
open index.html
```

> No build tools, no dependencies — just open and go! ✅
